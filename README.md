# ng-test
Angular 5.0 offical demo and unit and e2e test case.

## install packages
npm install

## install selenium web driver
npm run webdriver-update

## run demo
npm start

## run TypeScript lint
npm run lint

## run unit test
npm run test

## run unit coverage
npm run test-coverage

# run e2e test
npm run e2e

# run build
npm run build

# generate deploy files
npm run build-prod
